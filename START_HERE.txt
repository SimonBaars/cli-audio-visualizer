Your system has a monitor source available:
  alsa_output.pci-0000_65_00.6.analog-stereo.monitor

The easiest fix is to use pavucontrol:

  1. Install: sudo pacman -S pavucontrol
  2. Run the visualizer: python visualizer_stable.py
  3. In another terminal: pavucontrol
  4. Go to the "Recording" tab
  5. Find the Python/visualizer entry
  6. Change dropdown to: "Monitor of Analog Stereo"
  7. Watch the bars start moving! 🎵

Alternative - set as default:

    pactl set-default-source alsa_output.pci-0000_65_00.6.analog-stereo.monitor
    python visualizer_stable.py
